## 2. 字体系统 / Typography

### 字体家族
```
中文：PingFang SC, -apple-system, BlinkMacSystemFont, "Segoe UI", Microsoft YaHei, sans-serif
英文：Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif
代码：JetBrains Mono, Fira Code, Consolas, monospace
```

## 7. 组件规范 / Components

### 按钮

**主要按钮**
- 背景：Primary-500
- 文字：#FFFFFF
- 圆角：md (8px)
- 内边距：12px 24px
- 字体：Body, Semibold
- Hover：Primary-600
- Active：Primary-700
- Disabled：背景 Primary-200，文字 Primary-400

**次要按钮**
- 背景：透明
- 边框：Primary-500
- 文字：Primary-500
- 圆角：md (8px)
- Hover：背景 Primary-50

**幽灵按钮**
- 背景：透明
- 文字：Primary-500
- Hover：背景 Primary-50

### 输入框
- 高度：40px
- 内边距：12px 16px
- 边框：Neutral-200
- 圆角：md (8px)
- 字体：Body
- Focus：边框 Primary-500，阴影 ring Primary-200
- Error：边框 Error-500

### 卡片
- 背景：Neutral-0
- 圆角：lg (12px)
- 阴影：shadow-md
- 内边距：24px
